Arrays

  1. Write the code to swap the 5th element with the 7th element in an array of doubles called "stuff".

  2. Write the code to swap all odd elements with the even elements in a String array called "billygoat". The array has an even number of elements. Swap the first with the second, and the third with the fourth, and so on.

  3. Write the code to find the largest value in an array of doubles called "charlie".

  4. Write the code to sum all the elements in an array of ints called "doris".

  5. Write the code to find the first index where the element is equal to false in an array of booleans called "switchs".
