Variables

In the following, assume a is a double

  1. What does the following statement print?
    int w = -3;
double z = 6.0;

a = w * z + 3.4;

  2. What does the following statement print? 
    int w = 5;
int x = 14;
double y = 10.3;

a = x % w + 2 * y;


  3. What does the following statement print? 
    int w = 4;
double z = 2.0;

a = ((int)5.8 % (int)7.9 + w) / z;

  4. What does the following statement print? 
    int w = 12;
int x = 9;
double y = 2.3;

a = 5 * (w - x) - (int)5.2 % (int)y;

  5. What does the following statement print? 
    double y = 10.6;
double z = 13.3;

a = 6.8 / (2 + (int)y * (int)z % (int)y);
